La ligne de commande

Gestion de documents

  • Pour créer un dossier je dois utiliser cette commande :mkdir nomdudossier.
  • Pour voir où je me trouve : pwd.
  • Pour voir ce qu'il y a dans le dossier où je me trouve : ls + a
  • Pour les fichiers cachés + l
  • Pour la version longue.
  • Pour me déplacer dans un dossier : cd chemindudossier OU cd ..
  • Pour remonter d'un dossier.
  • Pour me mettre en super-utilisateur : sudo
  • Pour juste la ligne, sudo -s
  • Pour le rester, et sudo -i
  • Pour rester super ET se déplacer dans le dossier super utilisateur.
  • Pour mettre a jour la liste des paquets: sudo apt update.
  • Pour MAJ les paquets : sudo apt upgrade.
  • Pour installer un truc : sudo apt install truc.
  • Pour rechercher un mot/phrase : apt search mot/phrase.
  • Pour lancer l'installation d'un fichier :sudo dpkg -i nomdufichier.
  • Pour créer un fichier :touch nomdufichier.format.
  • Pour renommer un fichier :mv nomdufichier nouveaunom.
  • Pour déplacer un fichier :mv nomdufichier ~/chemindudossier.
  • Pour écrire dans un fichier : nano nomdufichier.
  • Pour lancer un fichier sur internet (genre .html) :firefox truc.html OU x-www-browser truc.html Pour navigateur par défaut.
  • Pour créer une liste : <ul> <li>élément 1 </li> <li> élément 2 </li> </ul>
